## Added to the collection

### Recurrent neural networks approximate continuous functions

This paper asks whether improving approximation accuracy always requires choosing a larger or newly parameterized neural network. Its answer is no: for every continuous function on `[-1,1]`, the authors prove that one fixed ReLU recurrent neural network with fixed hidden dimension can approximate the function uniformly, with better accuracy obtained by letting the same recurrent dynamics run longer.

Why it was added: the central object is a shared recurrent transition reused over time. The paper is not about repeated prompting, external agent loops, or autoregressive decoding; it treats the repeated application of one fixed-weight recurrent network as the computational resource. That makes it a theory-side flat-loop entry for the collection.

Main reader takeaway: the result frames runtime as an unavoidable approximation resource in a fixed-network setting. This complements looped-transformer and recurrent-depth work by giving a clean theoretical example where repeated shared computation, rather than additional parameters, buys accuracy.
