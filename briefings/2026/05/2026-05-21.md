# Daily Loop-Model Briefing — 2026-05-21

## Outcome

Four high-confidence papers were added to the main collection. The strongest architecture hit is **LT2: Linear-Time Looped Transformers** (`2605.20670`), which directly targets looped transformers and replaces quadratic attention with linear or sparse attention inside the repeated looped computation. The strongest reasoning hit is **Equilibrium Reasoners** (`2605.21488`), which treats scalable latent reasoning as repeated learned dynamics converging to attractors. The watch also added **Interaction Locality in Hierarchical Recursive Reasoning** (`2605.20784`) as a mechanistic analysis paper for HRM/TRM-style recursive models, and **Nonlocal operator learning for fMRI encoding and decoding tasks** (`2605.20389`) as an application of fixed-point latent operator inference.

## Added to the collection

### LT2: Linear-Time Looped Transformers

LT2 is squarely inside the repository scope: it starts from looped transformers, where layers are iterated multiple times before decoding, and asks how to make that repeated computation cheaper. The paper studies linear-attention and sparse-attention variants, argues that looping enables iterative memory refinement and receptive-field expansion, and reports gains on controlled recall, state-tracking, and language-modeling tasks. It is categorized as **Architecture and Algorithm Designs**.

### Equilibrium Reasoners: Learning Attractors Enables Scalable Reasoning

Equilibrium Reasoners frame reasoning as a latent dynamical system: a learned update rule is applied repeatedly, and successful reasoning corresponds to convergence toward task-conditioned attractors. The paper scales the internal dynamics both by depth (more iterations) and breadth (multiple stochastic trajectories), reporting large gains on Sudoku-Extreme when unrolling to very large effective depths. It is categorized as **Architecture and Algorithm Designs** because the reader takeaway is a reusable latent-dynamics model family and scaling recipe.

### Interaction Locality in Hierarchical Recursive Reasoning

This paper is analysis-focused. It applies sparse-autoencoder feature ablations, activation patching, and related checks to HRM and TRM, measuring how local information writes accumulate across repeated recursive updates into broader solution structure. Because the contribution is a mechanistic measurement framework for existing hierarchical/recursive loop models, it is categorized as **Theoretical and Mechanical Analysis**.

### Nonlocal operator learning for fMRI encoding and decoding tasks

This paper is an application-focused implicit-layer hit. Its latent neural integral-operator framework performs fixed-point iterations in an auxiliary latent space, then decodes predictions for fMRI classification or stimulus prediction. The evidence is more domain-specific and exploratory than the looped-transformer papers, but the fixed-point iteration is central enough for inclusion under **Applications Focused**.

## Checked but not added

### TERDNet: Transformer Encoder-Recurrent Decoder Network for Scene Change Detection

TERDNet uses a recurrent 3-gate-GRU decoder for iterative scene-change-mask refinement. That makes it adjacent to loop-model applications, but the paper is framed as a task-specific recurrent decoder for scene change detection rather than a general loop-model architecture or implicit-layer method. It stays on the watchlist for now.

### Chronicle: A Multimodal Foundation Model for Joint Language and Time Series Understanding

Chronicle shares transformer blocks across language and time-series modalities, but the abstract describes cross-modal parameter sharing, not repeated reuse of the same learned block within one model forward process. It was not added.

### Older backlog signal

**Simply Stabilizing the Loop via Fully Looped Transformer** (`2605.18797`) is clearly relevant, but its v1 date is outside this daily watch window. It should be considered in a separate backfill rather than mixed into today's new-paper intake.
