# Daily Loop-Model Briefing — 2026-05-20

## Outcome

One high-confidence paper from the 2026-05-20 watch was added to the main collection: **i-DEQ: A stable inertial deep equilibrium model for image restoration** (`2605.19705`). The inclusion evidence is direct: the paper is built around Deep Equilibrium Models, where each prediction is defined by solving a fixed-point problem, and the proposed inertial update changes the repeated equilibrium iterations themselves.

## Added to the collection

### i-DEQ: A stable inertial deep equilibrium model for image restoration

DEQs are already a canonical implicit-layer loop-model family: instead of stacking many explicit layers, they repeatedly apply a learned transformation until an equilibrium state is reached. This paper specializes that pattern to image restoration and inverse problems by learning an explicit nonconvex regularization inside the DEQ formulation and adding momentum to the fixed-point iterations.

Why it is in scope: the loop is not an agent loop, prompting loop, or repeated full-model call. The model's output depends on fixed-point inference in a learned implicit layer, and i-DEQ modifies that repeated internal computation with an inertial/momentum update.

Main reader takeaway: the authors report convergence guarantees and accelerated rates for the inertial fixed-point iterations. Empirically, i-DEQ is described as more stable during training and more robust to rough initialization than baseline DEQs, while achieving image-restoration quality comparable to state-of-the-art methods and reducing DEQ inference time by about a factor of two.

## Checked but not added

### Benchmarking and Evolving Reason-Reflect-Rectify for Reflective Visual Generation

This paper is relevant to iterative visual generation: it formalizes a Reason-Reflect-Rectify loop and benchmarks multi-round reflective generation. It was not added because the loop appears at the generation/workflow level, not as repeated reuse of the same learned internal layer, block, module, or fixed-point operator within one model forward process.

### Awakening the Hydra: Stabilizing Multi-Concept Backdoor Injection in Text-to-Image Diffusion Models

Hydra discusses open-source reuse, repeated downstream fine-tuning, and cumulative backdoor injection in diffusion-model ecosystems. That is outside the strict loop-model scope: the repeated process is checkpoint adaptation across parties/tasks rather than a single-forward reused learned operator.

### Real-Time Parallel Counterfactual Regret Minimization

Parallel CFR targets fast equilibrium solving for imperfect-information games. It was kept out because the iterations are an external algorithmic solver/pipeline, with neural inference used for leaf evaluation, rather than an implicit neural layer or shared learned operator reused to refine one prediction.
